# 📌 Changelog

Todas as alterações relevantes deste dataset serão documentadas aqui.

---

## [v1.0] – 2026-04-27
### Added
- Primeira versão do dataset S&P 500 em formato CSV.
- Inclui ticker, nome da empresa, setor, subsetor, localização e data de entrada no índice.
- Dados extraídos automaticamente da Wikipédia via Google Sheets.
