# 📊 Lista das Empresas do S&P 500  
![Dataset](https://img.shields.io/badge/Type-Dataset-0A66C2?style=flat-square)

Este repositório contém um ficheiro CSV com a lista atualizada das empresas que compõem o índice **S&P 500**, obtida diretamente da Wikipédia através do Google Sheets.

O objetivo é disponibilizar uma versão simples, limpa e facilmente reutilizável destes dados para projetos pessoais, análises ou estudos.
