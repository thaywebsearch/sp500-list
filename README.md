## 📘 Data Dictionary
| Coluna | Descrição |
|--------|-----------|
| **symbol** | Ticker da empresa |
| **name** | Nome da empresa |
| **sector** | Setor GICS |
| **sub_industry** | Sub‑indústria GICS |
| **headquarters** | Localização da sede |
| **date_added** | Data de entrada no índice |
| **cik** | Central Index Key (SEC) |
