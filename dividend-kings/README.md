# Dividend Kings Dataset (CSV)
![Release](https://img.shields.io/badge/release-v1.0-0A66C2?style=flat-square)

## 📊 Conteúdo do dataset
O ficheiro `dividend_kings.csv` inclui as seguintes colunas:

- **ticker** — símbolo da ação  
- **company** — nome da empresa  
- **sector** — setor de atividade  
- **years_increasing_dividends** — anos consecutivos de aumento de dividendos  
- **source_confirmation** — número de fontes independentes que confirmam a empresa como Dividend King
