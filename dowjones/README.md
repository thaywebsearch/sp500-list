# 📊 Dow Jones Industrial Average (DJIA) — Lista Completa das Empresas

![Version](https://img.shields.io/badge/version-v1.0-blue.svg)

Dataset com os 30 componentes do **Dow Jones Industrial Average (DJIA)**, incluindo nome da empresa, ticker e setor.  
Ideal para análise financeira, estudos de mercado, projetos de machine learning e visualização de dados.

---

## 📁 Ficheiros incluídos

- `dowjones-table.csv` — tabela completa em formato CSV  
- `dowjones-table.md` — versão em Markdown para leitura rápida  
- `VERSION` — versão atual do dataset  
- `CHANGELOG.md` — histórico de alterações  

---

## 🧩 Estrutura da Tabela

Colunas incluídas:

- **company** — nome da empresa  
- **ticker** — símbolo de negociação  
- **sector** — setor GICS  

---

## 🔄 Atualizações

Este dataset pode ser atualizado manualmente ou integrado num fluxo automatizado semelhante ao utilizado no projeto S&P 500.
